# CSS STYLE GUIDE

## Architecture -> SMACCS

There are five types of categories:

1. Base
2. Layout
3. Module
4. State
5. Theme

## Naming Rules

Layout: Prefixed with 'l'
    `l-[class name]`

Module: The name of the module. Example: Splash Component
`.splash`

State: Prefixed with 'is'
    `is-[class name]`

Theme: Prefixed with 't'
    `t-[class name] `
